#### 使用说明

准备阶段：
1.准备初始库存完整数据：需要在UU跌涨页面填写好每个饰品的购入价格等信息。
2.准备token：在UU网页登录之后，按f12获取接口中的token。

项目启动后：
1.  使用认证接口，设置uu的token到本地数据库，以便后续fegin调用
2.  使用初始库存接口同步饰品到本地数据库
3.  定时任务会监控今日的购买订单、出售订单数据，自动同步饰品总成本、购买总金额、购买均价、出售总金额等数据
4.  使用盈利接口获取饰品的总盈亏
